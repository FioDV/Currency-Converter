# Currency Converter App

## Description

The Currency Converter App is a Java-based desktop application developed using IntelliJ IDEA. It allows users to convert amounts between different currencies using real-time exchange rates from an API (https://www.exchangerate-api.com/). In addition to currency conversion, the app keeps a history of conversions, including the date and time of each transaction.

This project was built to provide a simple and interactive way to perform currency conversions while keeping track of previous operations.

## Features

- Convert USD to Argentine Pesos, Brazilian Reals, and Colombian Pesos.
- Convert Argentine Pesos, Brazilian Reals, and Colombian Pesos to USD.
- Keep a conversion history with timestamp.
- Display formatted results with two decimal points.
- Fetch real-time exchange rates using an API.
- Interactive menu-driven interface for ease of use.

## Technologies Used

- Java 17+
- IntelliJ IDEA
- HTTP requests to a currency exchange API
- BigDecimal for precise currency calculations
